 🚀 TaskFlow – Smart Task Management System

A modern, full-stack task management application designed to streamline workflows, improve productivity, and manage team collaboration efficiently.

📌 Overview

TaskFlow is a user-friendly and scalable task management platform that allows users to:
- Organize tasks visually
- Track progress in real-time
- Collaborate with team members
- Manage deadlines effectively

Built with a clean UI and powerful backend, this application is ideal for both individuals and teams.

 ✨ Features

🔐 Authentication
- Secure login & registration
- (Optional) Google OAuth integration

 📊 Dashboard
- Overview of tasks and progress
- Visual statistics and summaries

📋 Task Management
- Create, edit, and delete tasks
- Assign tasks to users
- Set priorities and due dates

📌 Kanban Board
- Drag-and-drop task management
- Columns:
  - To Do
  - In Progress
  - Completed

👥 Collaboration
- Multi-user support
- Task assignment and tracking

🛠️ Tech Stack
